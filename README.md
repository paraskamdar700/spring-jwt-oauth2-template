# Spring Boot Security Auth Starter

A reusable Spring Boot authentication template with:

- JWT Authentication
- OAuth2 Login (Google)
- Role-Based Access Control
- Global Exception Handling
- Stateless REST Security
- Production-ready structure

## Tech Stack
- Spring Boot
- Spring Security
- JWT
- OAuth2
- JPA
- MySQL

## How to Run

1. Configure database in application.properties
2. Set OAuth2 credentials
3. Run application
